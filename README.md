# ChampQuest

ChampQuest este o aplicație web construită cu React și Vite.

## Development

Pentru a porni aplicația în modul de dezvoltare:

```bash
npm install

npm run dev
```

Aplicația va fi disponibilă în browser la:

http://localhost:5173

Pagina se va reîncărca automat când faci modificări în cod.

## Build pentru producție

Pentru a crea versiunea optimizată:

```bash
npm run build
```

Fișierele generate vor fi în folderul `dist`.

## Preview build

Pentru a testa build-ul local:

```bash
npm run preview
```
